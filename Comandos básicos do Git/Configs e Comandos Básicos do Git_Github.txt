Configurações fundamentais do Git

git config --global user.email "seu_email"  'Cadastra o seu email no Git.

git config --global user.name "nome_usuário" 'Cadastra um nome de usuário no Git


Comandos básicos do Git/Github

git add *  'Enviar todos os arquivos da pasta local para área de Stage do Git.

git commit -m "Digite o texto aqui"  'Gerar commit da alteração feita no arquivo.

git remote add origin link_do_repositorio 'Adiconar um determinado repositório do Github.

git status 'Verificar se houve alguma alteração nos arquivos

git remote -v 'Listar os repositórios criados e disponíveis do Github.

git push origin master 'Empurrar os arquivos locais para o servidor do Github.

git pull origin master 'Puxar os arquivos hospedados no repositório do Github.

git clone link_do_repositorio  'Clonar um determinado repositório.
